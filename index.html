Proyectos

  • Rescue App

    Se trata de una plataforma Web pensada para que en conjunto con una Aplicación móvil sirvan al Cuerpo de Bomberos de la Ciudad de Encarnación como medio para informarles de un accidente de tránsito.

    La plataforma Web fue desarrollada con PHP, HTML, JavaScript, CSS, Bulma, MySQL; en ella se reciben los datos enviados por los usuarios de la aplicación móvil, así también tendrán acceso a la ubicación geográfica de donde se emitió el reporte además de datos particulares del/a accidentado/a; el acceso está restringido solamente para los Bomberos.

    La Aplicación móvil fue desarrollada en su totalidad utilizando MyAppInventor, esta registra los datos del usuario, y una vez que el usuario emita un reporte Urgente o detallado, estos datos se envía a la plataforma web, para que los bomberos puedan tener acceso a los datos del/la accidentado/a, tipo de sangre, edad, nro de familiar, tipo de vehiculo con el que pudo haberse accidentado, entre otros, que pueden servir de ayuda a los Bomberos.

    Este proyecto se plantea como una alternativa a la tradicional "llamada" que se realiza para contactar con los Bomberos, pero caso el usuario no posea acceso a internet en el momento de emitir un reporte la aplicación intentará realizar una llamada directamente a los bomberos *132 para que el usuario aún así pueda contactar con éstos.

  • Farming Note

    Este proyecto no se dió seguimiento, sin embargo era un proyecto que pretendía reunir todos los datos de una Granja Agrícola de manera Digital, en otras palabras, los recursos utilizados para los distintos procesos que se realizan en los distintos cultivos y zafras durante la labor agrícola, y de esta manera tener un historial de procesos realizados para una gestión correcta de "rotación de cultivos" en las distintas parcelas, también se planteaba la idea de utilizar Google Maps para medir los terrenos y guardarlos como terrenos.
